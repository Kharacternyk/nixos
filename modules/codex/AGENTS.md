If a tool you need is unavailable,
neither attempt to install it yourself nor silently fall back to a less efficient workflow,
ask the human how to proceed instead.

Never silently overwrite changes that have been made to files since you last read them,
ask the human how to proceed instead.

When editing prose, keep each sentence on its own line,
unless the sentence is very short.
If a sentence is longer than 93 characters, hard wrap it.
Prefer line breaks at punctuation.
Prefer line breaks that do not introduce a very short line.
If it is clear that the prose in question follows other formatting rules,
obey those rules instead.
